CS Advanced Topics Projects

Magic Mirror Project

In Advanced Topics in Computer Science, me and a group of people are making a smart mirror. The mirror will have a monitor behind a mirror, displaying information onto the mirror while still being able to view yourself. A raspberry pi and associated accessories will be used to power the logic and code behind the mirror.

Timeline and planning sequence

Early Stages

  • Research & Conceptualization
    • Explored existing smart mirror designs, researched necessary hardware, and developed initial project goals.
  • Hardware Selection & Assembly
    • using the raspberry pi 5, we also purchased a camera, Micro HDMI to HDMI, and took a computer monitor from the comp sci room. USB C for power to the pi, and a sim card into the pi for internal storage of the Raspberry Pi OS used
  • Software Development & UI Design
    • Downloaded existing module functionality from a smart mirror framework open sourced from GitHub, and then use design elements from these modules to make our own School Period Tracker with a progress bar for the top of the mirror

Expanding Functionality

  • Integration of Real-Time Data
    • refined the school period tracker to have the Vernon Hills High School Schedule period times properly in the program, still have yet to figure out how to implement API into it to fetch data from multiple sets
  • AI & Machine Learning Enhancements
    • use OpenCV to implement facial recognition to create a profile system and have personalized information for each person that walks up to the mirror.
